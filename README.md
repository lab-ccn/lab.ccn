# Fedora Linux Ublue Lab Image

Fedora Kinoite based UBlue image with preinstalled software and restrictions for use in education.

## Additions

### Main Additions:

- Cisco Packet Tracer
- VirtualBox
- PicoCom

  
### Misc Additions

- LibreOffice
- Remmina
- PuTTY
- FileZilla
- hyfetch
- Btop
- Fastfetch
- Kitty
- Chromium
- Tmux
- Bat
